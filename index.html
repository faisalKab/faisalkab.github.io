Touchless FPGA Maze Game

Designed a real-time interactive maze game controlled entirely by hand gestures. Built on the DE10-Lite FPGA board, this project uses IR sensors to detect directional input and servo motors for dual-axis tilt. Integrated a timer and score-saving system with HEX display output. Written entirely in Verilog and implemented with physical motion components.
